# Drone-Based Cyclone Forecasting & Disaster Response
Research project exploring how autonomous drone systems can improve tropical 
cyclone forecasting and accelerate post-disaster response. Conducted at 
The Knowledge Society (TKS), Dubai.

---

## Overview
Current cyclone forecasting relies on satellites and ground-based radar, 
which struggle with low-altitude atmospheric data and rapid intensification 
events. This project investigates how drone swarms could fill that gap — 
collecting in-situ data in environments too dangerous for crewed aircraft.

## Key Questions Investigated
1. What data gaps exist in current cyclone forecasting models?
2. How could autonomous drones operate in pre-landfall cyclone conditions?
3. What is the forecasting improvement of adding low-altitude in-situ data?
4. How does this apply to disaster response (search and rescue, damage mapping)?

## Key Findings
- Low-altitude pressure and wind data is consistently underrepresented in 
  current operational models
- Expendable drone designs are more viable than recoverable ones in 
  Category 2+ conditions
- Post-disaster damage mapping is more immediately deployable than 
  pre-storm sensing
